Module B – CAD

CAD Data Processing & Validation

B.

Module B – CAD (CAD Data Processing & Validation)
Ensures the ingestion, validation, and structuring of native CAD files: geometry extraction, metadata, parts, assemblies, and technical layers. It performs automatic quality control and prepares the data for mapping and export steps.
MetricX's Module B is designed as an integrated CAD connectivity solution, supporting a wide range of CAD software and file formats for seamless data flows and interoperability.

  • Multi-CAD support: PTC Creo (Pro/ENGINEER Wildfire), CATIA, SolidWorks, Siemens NX, SolidEdge, Autodesk Inventor…
  • Compatible file formats: .prt, .asm, .drw, .catpart, .catproduct, .sldprt, .sldasm, .dxf, .step (.stp), .iges, .json
  • Component decomposition & validation: extraction of parts, sub-assemblies, profiles, features, business attributes.
  • Automatic geometric and attribute consistency checks (business rules, client rules).
  • Entity enrichment: UID assignment, normalization, BOM mapping, and technical metadata export.
  • Export and integration: DXF, STEP, and enriched JSON for downstream processing and interoperability with PLM/PDM (e.g., Windchill, Teamcenter, 3DEXPERIENCE, SAP PLM, etc.).

Module C – DXF Extract

Automated Extraction of 2D Entities

C.

Module C – DXF Extract (2D Entity Automation)
Automates the analysis of 2D drawings (DXF: AutoCAD, DraftSight, etc.) to extract all technical entities (lines, blocks, layers, annotations, and more). Each extracted entity is validated, named, and converted to JSON for subsequent 3D mapping or XR export steps.






  • Intelligent DXF parsing: layers, blocks, polylines, texts, attributes.
  • Advanced OCR for scanned drawings or non-standardized texts.
  • Validation & enrichment: geometry checks, business entity detection (zones, dimensions, symbols, etc.).
  • Standardized JSON export for every detected entity (ready for mapping or control AI).

Module D – STEP Map

2D to 3D Mapping & Data Association

D.

Module D – STEP Map (3D Digital Thread Automation)
Generates an enriched and automated mapping between detected 2D entities (from Module C) and 3D CAD model objects (STEP, IFC, native). This module produces a file specially formatted for process automation performance and advanced XR capability, with robust XYZ coordinate management—particularly on the Z-axis. It guarantees a complete digital thread, optimal traceability, and direct data exploitation for XR, digital twin, and simulation.
Example: part M0165-05-0001-01 — advanced analysis and communication on the Z axis for automated integration and technical compliance.

  • Advanced reading and parsing of STEP, IFC, 3D XML files…
  • Automated 2D/3D matching: associations between DXF layers from Module C, CAD features, and 3D objects.
  • XYZ axis and coordinate management: emphasis on Z-axis robustness and accuracy.
  • Validation, audit, and generation of enriched JSON mapping, ready for XR and 3D simulation.
  • Advanced communication: interoperability, automated logs, and reporting for PLM/ERP/XR integration.

Module E – XR

XR/3D Export & Visualization

E.

Module E – XR (XR/3D Export & Visualization)
Transforms all validated and mapped data into interactive 3D objects, exportable to XR platforms (Unity, Unreal Engine, WebXR, Blender, TouchDesigner). Module E enables immersive visualization of digital twins, real-time simulation, and on-site validation for engineering and production.
MetricX’s XR module is designed for direct and structured communication with leading 3D engines, leveraging enriched JSON for maximum efficiency and productivity throughout the digital pipeline.

  • Automated generation and export of 3D models + metadata to XR/AR/VR platforms.
  • Streaming of technical attributes: colors, materials, structures, tolerances, and more.
  • Native compatibility: Unity3D, Unreal Engine, Blender 3D, TouchDesigner, WebXR, and cloud XR platforms.
  • Live export monitoring, performance logs, and integrity auditing for robust deployment.
  • Audit-ready JSON output, optimized for seamless ingestion by 3D engines and business applications.
